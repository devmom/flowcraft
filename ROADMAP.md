# FlowCraft Roadmap

## v0.1.0 (Current — MVP)

- [x] Model Gateway (DeepSeek, Agnes AI, Ollama)
- [x] Intent Engine (task type classification)
- [x] Planner (Direct, Linear, DAG modes)
- [x] Execution Engine with checkpoints & recovery
- [x] Policy & Approval Engine
- [x] Tool System (15+ built-in tools)
- [x] Workflow Builder
- [x] Memory System (short-term, long-term, knowledge base)
- [x] Secret Store (encrypted API key storage)
- [x] Plugin Architecture
- [x] Web UI
- [x] Observability (trace events, replay)

## v0.2.0 (Planned)

- [ ] Streaming LLM responses (SSE)
- [ ] Multi-agent collaboration
- [ ] Conditional branching in workflows
- [ ] Retry policies with exponential backoff
- [ ] Webhooks for task lifecycle events
- [ ] Session management & multi-user support
- [ ] Docker support
- [ ] macOS/Linux portable builds

## v0.3.0 (Planned)

- [ ] Visual workflow editor (drag-and-drop)
- [ ] Agent marketplace / plugin store
- [ ] RAG pipeline (document indexing + retrieval)
- [ ] Multi-modal support (image understanding)
- [ ] Scheduled tasks & cron-like triggers
- [ ] Export/import workflows as JSON/YAML
- [ ] Performance benchmarking suite

## v1.0.0 (Future)

- [ ] Stable API (backward compatibility guarantee)
- [ ] Comprehensive test coverage (>80%)
- [ ] Full documentation site
- [ ] Community plugins ecosystem
- [ ] Enterprise features (RBAC, audit logs, SSO)
- [ ] gRPC API alongside REST
- [ ] Desktop application (Tauri/Electron)

## Ideas & Backlog

- OpenTelemetry integration
- LangChain / LlamaIndex interop (optional bridge adapters)
- Voice interface
- Mobile companion app
- One-click deploy templates (Vercel, Railway, etc.)
